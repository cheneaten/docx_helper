🎛️ 标题格式自定义

  • 在左侧 配置管理 面板可修改各级标题的编号样式（如改为 「一」 等）
  • 取消勾选 "自动编号" 可隐藏所有标题编号
  • 编号随标题增删和层级变化自动重新计算
  • 正文默认字体和字号也在配置面板中修改

📝 正文格式

  • 默认正文：宋体 / 小四（12pt）/ 行距 1.5
  • 使用 Ctrl+B / Ctrl+I / Ctrl+U 快速加粗、斜体、下划线
  • 选中文字后可用工具栏或右键菜单调整字体、字号、颜色

🖱️ 打开图片编辑器

  • 在编辑器中的图片上 右键点击，弹出菜单选择：
  • 🔍 查看图片 — 查看模式，支持缩放和拖拽
  • ✏️ 编辑图片 — 编辑模式，可添加标注
  • ✂️ 裁剪图片 — 裁剪模式，选区裁剪

🔍 查看模式

  • 滚轮缩放：鼠标滚轮放大/缩小图片（10%~500%）
  • 拖拽平移：放大后按住鼠标拖拽可平移查看
  • 适应窗口：点击工具栏 适应窗口 按钮或按 0 将图片缩放到窗口大小
  • 工具栏按钮：放大 + / 缩小 - / 适应窗口

✏️ 编辑模式 — 绘图工具

  • 🖊️ 画笔 (P)：自由绘制线条，可调颜色和线宽
  • 🧹 橡皮 (E)：擦除标注（擦除与路径相交的标注对象）
  • 📝 文本框 (T)：点击图片放置文字，支持中文输入
  • ⬜ 方框 (R)：绘制矩形框，可用于标注重点区域
  • 📏 直线 (L)：绘制直线
  • ➡️ 箭头 (A)：绘制箭头，指向关键位置
  • ✂️ 裁剪 (C)：进入裁剪模式，拖拽选择区域

🎨 工具栏功能

  • 颜色选择器：设置画笔、方框、直线、箭头的颜色
  • 线宽滑块：设置画笔和线条的粗细（1~20px）
  • 撤销/重做：Ctrl+Z / Ctrl+Y，最多 50 步
  • 缩放：+ / - / 适应窗口（快捷键 + / - / 0）

✂️ 裁剪模式

  • 在图片上 拖拽鼠标 框选需要保留的区域
  • 选区四角和四边有 8 个拖拽手柄，可调整选区大小
  • 选区内部可 拖拽移动 整个选区位置
  • 辅助线：九宫格虚线 帮助构图
  • 点击底部 ✅ 确认裁剪 完成裁剪
  • 点击 ❌ 取消 或按 Esc 退出裁剪

💾 应用与放弃

  • ✅ 应用修改：将编辑后的图片（含标注/裁剪结果）写回编辑器，触发自动保存
  • ❌ 放弃修改：丢弃所有标注和裁剪，恢复原始图片
  • 修改后的图片以 base64 格式嵌入文档，导出 DOCX 时一并包含

📌 锚点功能

  • 添加锚点：将光标放在段落中，按 F9 或在左侧装订线点击目标段落位置
  • 移除锚点：再次按 F9 或点击装订线中的红色圆点
  • 锚点面板：点击工具栏 📌 锚点 按钮打开面板，查看所有锚点列表
  • 跳转：在面板中点击 跳转 按钮，或使用 Ctrl+Shift+↑/↓
  • 快捷键导航：面板中 上一个/下一个 按钮逐一跳转，目标段落会高亮闪烁
  • 清除全部：面板底部提供一键清除所有锚点
  • 锚点数据随文档保存到 IndexedDB，刷新后自动恢复
  • 装订线在编辑器纸张左侧，锚点段落旁显示 红色圆点标记

📁 折叠功能

  • 添加折叠区域：在任意两个段落分别设置锚点，即形成一段可折叠区域
  • 折叠/展开：点击装订线中两个锚点之间的 括号标记 【 切换折叠状态
  • 折叠面板：点击工具栏 📁 折叠 按钮，查看所有折叠区域并管理
  • 折叠后可隐藏中间段落内容，便于专注编辑其他部分
  • 折叠数据随文档保存到 IndexedDB

📑 目录导航

  • 右侧 目录 (TOC) 面板自动从标题生成层级结构
  • 点击目录中的标题可直接 滚动跳转 到对应位置
  • 目录随编辑器内容变化实时更新

🚀 快速上手

  1. 点击 📂 导入 DOCX 或 Ctrl+O 打开一个 Word 文档
  2. 直接在编辑器中编辑，格式自动保留
  3. 点击 💾 导出 DOCX 或 Ctrl+Shift+S 保存结果
  4. 编辑过程中数据 自动保存 到浏览器本地存储

💡 进阶功能

  • 🖌️ 格式刷：点击格式刷按钮后，先点击源段落复制格式，再点击（或框选）目标段落应用
  • 🤖 AI 标题助手：将文档内容与 format-rule.md 一起发给大模型，把返回的 JSON 粘贴到助手面板，自动在锚点位置插入标题
  • 😊 表情栏：点击工具栏表情按钮，在光标位置插入 emoji，可在左侧面板自定义分类和常用表情
  • 💻 代码块：支持语法高亮和行号，导入 DOCX 时 1×1 表格自动转为代码块
  • 📊 表格：支持单元格合并、行列增删、对齐和边框设置，右键表格可快速操作
  • 🎨 自由绘图：点击工具栏 🎨 绘图按钮进入画板模式，支持形状、流程图、思维导图
  • 🐾 陪伴宠物：编辑器右下角有三只可选宠物（🐱小橘猫、🐶旺财狗、🐰雪球兔），会响应写作节奏产生不同反应、随机鼓励气泡；可在右侧面板切换宠物和编辑宠物档案（姓名、年龄、生日、身高、体重、学历、爱好等）
  • 🖨️ 打印：点击打印按钮生成适合打印的版式页面

💾 数据存储与备份

  • 主存储：基于浏览器 IndexedDB，数据完全本地，无需网络
  • 自动保存：每次输入后 2 秒防抖自动保存；每 30 秒周期保存
  • 关闭保护：关闭或刷新页面时弹出离开确认对话框，同步写入 localStorage 兜底
  • 恢复机制：重新打开页面时自动从 IndexedDB 恢复，失败则从 localStorage 恢复
  • 多标签存储：最多 3 个文档标签页，每个标签文档独立保存

⚙️ 配置管理

  • 左侧面板可调整 标题格式、正文字体、字号、行距
  • 点击 💾 导出配置 将当前设置保存为 docx-editor-config.json 文件
  • 点击 📂 导入配置 从文件恢复配置
  • 将配置文件放在项目目录下，启动时自动读取

🎨 界面与环境

  • 🌙 暗色主题：点击工具栏月亮按钮切换亮/暗主题，适合夜间写作
  • ⛶ 全屏模式：按 F11 进入浏览器全屏，沉浸式编辑
  • 📊 实时统计：底部状态栏显示字数、行数、图片数、代码块数、表格数、文件大小、编辑时长
  • 右键菜单：编辑器内右键提供快捷操作（表格编辑、图片编辑、文本格式等）
  • 🔍 查找替换：Ctrl+F 查找，Ctrl+H 查找并替换，支持大小写敏感

📄 文件兼容性

  • 支持导入 .docx 格式（基于 mammoth.js）
  • 导出为标准 Office Open XML 格式，兼容 Microsoft Word / WPS
  • 图片以 base64 嵌入文档，无需外部资源文件
  • 支持 表格、代码块、标题层级、图片标注 的完整导入→编辑→导出往返

⌨️ 提示

按 F1 随时打开本帮助弹窗。帮助弹窗可 拖拽标题栏 移动位置，不会被其他窗口遮挡。更多快捷键请查看 ⌨️ 快捷键 标签页。
